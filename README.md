# AI Code Reviewer

An AI-powered code review application built using React, Vite, Node.js and Express that analyzes code snippets and provides structured feedback similar to a professional developer review.

## Features

* AI-generated code reviews
* Suggestions for code optimization
* Best practice recommendations
* Edge case handling suggestions
* Cleaner and maintainable code improvements
* Syntax highlighted code blocks
* Responsive developer-friendly UI

## Tech Stack

### Frontend

* React
* Vite
* CSS

### Backend

* Node.js
* Express.js

### AI Integration

* Gemini / OpenRouter API

## How It Works

1. User writes or pastes code into the editor.
2. Frontend sends the code to the backend API.
3. Backend forwards the prompt to the AI model.
4. AI analyzes the code and generates:

   * Code quality feedback
   * Optimization suggestions
   * Error handling improvements
   * Best practice recommendations
5. The reviewed response is rendered dynamically in the UI.
